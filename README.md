# ✈️ ixigo Clone

A frontend clone of the popular travel booking platform, **ixigo**. This project focuses on replicating the clean, user-friendly user interface (UI) of ixigo's homepage, including search forms for flights, trains, and buses.

## 🚀 Live Demo
Check out the live interactive preview of the project here:
🔗 **[Live Demo Link](https://polojuarunkumar.github.io/IxigoApplication/ixigo/indigoFlight.html)**

---

## ✨ Features
* **Interactive Search Tabs:** Seamless switching between Flights, Trains, Buses, and Hotels.
* **Modern UI Layout:** Clean navigation bars, promotional banners, and structured booking forms mimicking the original site.
* **Responsive Design:** Optimized layout components to ensure standard web compatibility.
* **Custom Styling:** Tailored CSS for accurate color schemes, typography, and button states.

---

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for booking forms, inputs, and layout.
* **CSS3:** Custom layouts, flexbox positioning, hover effects, and brand color consistency.
